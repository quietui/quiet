Focus on creating, not recreating. The Quiet Figma Starter kit will save you months of work to create a single source of truth between design and engineering for the most commonly used components. Improving design to development workflows, helping teams reach quality outcomes, and delivering value faster.

The Quiet Figma Starter Kit is built and maintained by [Ange Romanska](https://www.linkedin.com/in/angeromanska/).

## TL;DR

- Mirrored Figma and Web components
- Fully customizable with primitive and adaptive variables
- Semantic and responsive ready type styles
- Light and dark modes
- Prebuilt style variants with nested instances, and flexible property configuration
- Getting started documentation to get you up and running in no time
- Component usage guidelines

## Who it's for


The Figma Starter Kit is built to flex with your needs. Whether you're spinning up a quick proof of concept using ready-to-go components and default themes, or building a fully customized, brand-aligned design system that scales across teams and platforms. It supports simple use cases with minimal setup, while also offering the structure, tokens, and styling capabilities required for complex, multi-theme implementations.

## What's included

When you purchase the Quiet Figma Starter Kit you will be given access to download the Foundations, Icon Library, and Component Library Figma (.fig) source files. Allowing you full control to style, customize, and evolve the system to meet your brand and use case needs.

### Foundations

<img src="/assets/images/figma/foundations.svg" alt="TODO">

The Foundations file is used to define the variable driven styling that applies to the components. Allowing you to easily apply your preferred colors, typography, shadow effects, corner radius and more in one place. This file was created separately to allow library owners to restrict editing permissions of the variables if needed.

### Icon library

<img src="/assets/images/figma/icons.svg" alt="TODO">

The icon library includes more than 400 frequently used icons from the open source Tabler icon set. With clear guidance on how to add new instances from the Tabler Figma plugin, or add your own custom icons to the set.

### Components

<img src="/assets/images/figma/components.svg" alt="TODO">

The Figma component library matches the full set of web components including variants for size, appearance, state, orientation and more. Built with nested instances that use the same tokens and naming structure, you can use them as is, or easily tailor them to your needs without introducing unnecessary complexity during development. 

Usage guidelines for when to use and when not to use each component have also been included to help newer designers learn, and provide a foundation for governance if needed.
